# Usage

## Example

Run

```bash
# gs_usb
./server.py run \
    --interface gs_usb \
    --srv-interface yachtd_raw \
    --channel "" \
    --port 1000 \
    --log-level INFO 

# slcan
./server.py run \
    --interface slcan \
    --srv-interface yachtd_raw \
    --channel "/dev/tty.usbmodem1101" \
    --port 1000 \
    --log-level INFO 
```
